1. Выбор города
	Пользователь нажимает на инпут для выбора города.
	Из подгрузившегося списка выбирает тот, из которого он отправляется.
	Из подгрузившегося списка городов для определенной местности выбирает город отправления.
Выбранный город вставляется в инпут.

2. Выбор даты
	Пользователь нажимает на datepicker.
	Выдит календарь с двумя месяцами (текущий и следующий).
	Из календаря стрелками можно поменять месяцы, но не старше, чем текущий, а также ограниченный одним годом вперед.
	Выбираем нужную нам дату.
Дата вставляется в инпут.

3. Работа с multy city
	Пользователь нажатием кнопкой "Добавить" он добавит себе полет (максимум 6).
	Пользователь может удалить полет кнопкой "Удалить" (должно остаться минимум 2).
	В каждом полете выбрать город отправления, город посещения и дату отправки.

4. Букинг билета
	Пользователь выбирает вкладку Book Flights.
	Выбирает способ полета (return, one way, multy city).
	Выбирает город, из которого будет вылет (use-case 1).
	Выбирает город, в который летит (use-case 1).
	Если был выбрын return, то выбирает дату отправления и возврата (use-case 2); если one way, то только отправления (use-case 2);
если же был выбран multy city, то use-case 3.
	Выбрать количество билетов для взрослых (1 - 10).
	Количество детей (0 - 6).
	Количество младенцев (0 - 2).
	Выбрать тип полета (Economy / Business).
	Нажать на кнопку "Найти".
Пользователь увидит сообщение о завершении и результаты поиска.

5. Найти машину
	Пользователь выбирает страну, где заберет машину.
	Выбирает город, где заберет машину.
	Выбирает дату (use case 2).
	Выбирает время (часы и минуты).
	Выбирает оставит он машину там же, где и взял или нет.
	Если выбрано не там, где взял - то пользователь повторяет действия 5.1 - 5.4, но только для места, где он оставит машину.
	Выбирает город пребывания.
	Выбирает тип машины (mini, exotic, lusury и т.д.).
	Нажимает на кнопку "Найти".
Пользователь видит результаты поиска.

6. Бронирование отеля
	Пользователь выбирает страну пребывания.
	Выбирает город пребывания.
	Выбирает дату прибытия.
	Выбирает дату отъезда.
	Выбирает количество комнат.
	Выбирает количество взрослых людей.
	Выбирает количество детей.
	Выбирает количество младенцев.
	Нажимаеи на кнопку "Найти".
Видит результаты поиска.

7. Просмотр расписания полетов.
	Пользователь заходит на страницу с расписанием.
	Выбирает тип полета (one way, return).
	Из списка выбирает город отправления.
	Выбирает город прибытия.
	Выбирает дату отправления.
	Если был выбран тип return, то дату возврата.
	Выбирает показывать ли только прямые рейсы или все.
	Выбирает показывать таблицу с расписанием на неделю или на день.
	Нажимает на кнопку "Посмотреть расписание".
Пользователь видит список рейсов для выбраных данных.

8. Онлайн регистрация на рейс.
	Пользователь входит на страницу для онлайн регистрации.
	Вводит номер билета или ссылку на букинг.
	Вводит свою фамилию.
	Вводит аэропорт отправки.
	Нажимает кнопку "Получить".
Пользователь видит сообщение об успешной регистрации или сообщение об ошибке.

9. Проверка рейса
	Пользователь вводит номер рейса.
	Выбирает дату отправки.
	Нажимает кнопку "Проверить".
Видит информацию о рейсе или сообщение об ошибке.

10. Просмотреть свои бронирования полетов.
	Пользователь вводит свою фамилию.
	Вводит ссылку на бронирование.
	Нажимает кнопку "Получить".
Видит информацию по введенным данным или информацию об ошибке.
